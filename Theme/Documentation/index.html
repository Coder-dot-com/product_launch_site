Customization

How to change Logo:

Answer: Open your index.html file and go to code line 81, change the image source as you want.



How to change Footer Copyright:

Answer: Open your index.html file and go to code line 987, change the Copyright as you want.



How to make header nav sticky:

Answer: Open your index.html file and go to code line 75, add a id #tj-header-sticky to header tag.

Contact Form

How to change Email address of "Contact Form":

Answer: Open your /assets/mail.php file and go to code line 18, change the Email address as where you want to get the contact form submitted.
